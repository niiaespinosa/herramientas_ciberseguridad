## Glosario de Conceptos Básicos

*   **Ciberseguridad**: Es la práctica de proteger sistemas, redes y programas de ataques digitales destinados a acceder, modificar o destruir información confidencial.
*   **Phishing**: Técnica de engaño donde un atacante se hace pasar por una entidad legítima para robar datos personales o bancarios a través de correos, mensajes o sitios web falsos.
*   **Malware**: Término genérico para cualquier software malicioso (virus, gusanos, troyanos) diseñado para infiltrarse o dañar un dispositivo sin el consentimiento del usuario.
*   **Vulnerabilidad**: Un punto débil o falla en el diseño o configuración de un sistema que un atacante puede aprovechar para comprometer la seguridad.
*   **Autenticación**: Proceso para verificar la identidad de un usuario, dispositivo o sistema (por ejemplo, mediante contraseñas o datos biométricos).
*   **Autenticación de Dos Factores (2FA)**: Capa de seguridad adicional que requiere dos formas diferentes de identificación para acceder a una cuenta (como una contraseña y un código enviado al celular).
*   **Cifrado (Encriptación)**: Proceso de convertir información en un código secreto para evitar que personas no autorizadas puedan leerla o acceder a ella.
*   **Ingeniería Social**: Manipulación psicológica de las personas para que realicen acciones o revelen información confidencial, aprovechándose de la confianza humana en lugar de fallas técnicas.
*   **Riesgo**: La probabilidad de que una amenaza aproveche una vulnerabilidad para causar un impacto negativo en un activo de información.
